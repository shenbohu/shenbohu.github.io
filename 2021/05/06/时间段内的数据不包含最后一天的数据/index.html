时间段内的数据不包含最后一天的数据

获取一段时间段内的数据不包含最后一天的数据
1.用 StartTme < date < endTime
这种方法starttime和endtime 可以不传
2 用 between StartTme and endTime
这种的starttime和endtime 必须要有 缺一不可
但是这2种方法都获取不到 endTime 当天的数据 改成<=也不行
我自己想的是可以吧endTime的时间加一天 用新的查询
public static Date getNextDay(Date date) {
Calendar calendar = Calendar.getInstance();
calendar.setTime(date);
calendar.add(Calendar.DAY_OF_MONTH, +1);//+1今天的时间加一天
date = calendar.getTime();
return date;
}

自己只能想到这种方法 有好的方法可以回复
